 # Basics of Web Development

## 📖 Project Overview
This project is a beginner-friendly introduction to **Web Development** using **HTML, CSS, and JavaScript**. It demonstrates the fundamentals of creating a simple webpage, styling it with CSS, and adding basic interactivity using JavaScript.

## 🎯 Objectives

- Understand the basic structure of an HTML document.
- Learn how to style web pages using CSS.
- Implement simple interactivity with JavaScript.

---

## 📌 Features

### 1. Create a Simple Webpage (HTML)
- Added headings and paragraphs.
- Inserted images.
- Added hyperlinks for navigation.

### 2. Style the Webpage (CSS)
- Applied colors and fonts.
- Added spacing and margins.
- Improved layout and appearance.

### 3. Add Interactivity (JavaScript)
- Created a button.
- Displayed an alert message when the button is clicked.

## 🛠️ Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript (ES6)**
